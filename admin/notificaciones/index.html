Cómo funciona

1
Selecciona el grupo de clientes y escribe el mensaje
2
Haz clic en "Preparar mensajes" — se genera un botón por cliente
3
Haz clic en cada botón verde — se abre WhatsApp con el mensaje listo
4
Solo das clic en "Enviar" en WhatsApp — sin costo, sin API
✓ Funciona con cualquier número de WhatsApp sin necesidad de API ni pagos mensuales.
